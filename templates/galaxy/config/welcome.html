The Vertebrate Genome Lab (VGL) focuses on generating high-quality reference genomes for a variety of Earth's biodiversity, as part of the larger Earth Biogenome Project. Check out our paper for more information about the pipeline!

There are two tutorials available for learning how to use the VGP pipeline:
  • The longform, step-by-step tutorial with explanations of assembly concepts and ideas, including rationales for various parameters. If you are new to genome assembly, check this one out first and follow along with test data!
  • The shorter tutorial meant to show how to run our workflows. If you already have data in-hand and are ready to run our workflows (i.e., NOT running the pipeline step-by-step), then check this one out!
